roobot

a discord bot with activities and whatever

invite coming never

features

  • rarely gets updated
  • runs on a raspberry pi
  • restarts every 24 hours to refresh commands
  • code quality that may be questionable at best

commands

/quote

number: int

text: bool

activity

random quote thats "inspirational"

variables

number: choose a certain quote

text: changes quote from embed to text

/8ball

question: str

text: bool

activity

either yes or no

variables

question: the message/question for 8ball

text: changes 8-ball result from embed to text

/tabletennis

opponent: usr

activity

beta

play table tennis

variables

opponent: the player to go against

/giveaway

utility

subcommands

manage giveaways

subcommands

create

item: str

end_time: int

winners: int

channel: channel

end_embed: bool

create a new giveaway

item: the item to giveaway

end_time: the time in seconds to end

winners: amount of winners allowed

channel: the channel to send the giveaway prompt to

end_embed: whether to use an embed or plain text at giveaway end

builder

gui menu to create a giveaway in current channel

reroll

id: str

amount: int

end_embed: bool

choose another winner for a giveaway

id: the id of the giveaway to reroll

amount: how many should be rerolled

end_time: whether to use an embed or plain text at reroll

/poll

question: str

option1: str

option2: str

option3: str

option4: str

utility

create a poll

variables

question: the question/message for poll

option1: the first option for poll

option2: the second option for poll

option3: the third option for poll

option4: the fourth option for poll
